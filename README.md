
# Single page – API JSONPlaceholder

Projeto introdutório do Curso de React.Js e Next.Js (Intermediário e Avançado).



## Instalação

No diretório do projeto, instale as dependências: 

```bash
`npm install`
```

Testes (JEST):
```bash
`npm test`
```

## Stack utilizada

React (Hooks), Node, Jest, JSONPlaceholderAPI
